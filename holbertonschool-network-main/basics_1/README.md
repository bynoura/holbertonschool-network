# Networking Basics #1

This project covers practical networking tasks including configuring `/etc/hosts`, displaying IP addresses, and listening on ports.

## Tasks

### 0. Change your home IP
Write a Bash script that configures an Ubuntu server to change localhost to resolve to 127.0.0.2 and facebook.com to resolve to 8.8.8.8.

### 1. Show attached IPs
Write a Bash script that displays all active IPv4 IPs on the machine.

### 2. Port listening on localhost
Write a Bash script that listens on port 98 on localhost.
